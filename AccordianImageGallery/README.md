# Image Gallery

This is a responsive and interactive image gallery application built with HTML, CSS, and JavaScript, enhanced with Bootstrap for a modern and clean design. It allows users to browse, search, upload, randomize, and view images in a sleek lightbox.

## Features

   Responsive Design: Optimized for various screen sizes, from mobile devices to large desktops.
   Dynamic Image Loading: Images are loaded progressively, with a "Load More" feature to fetch additional content.
   Image Search: Filter images by their descriptions using a real-time search bar.
   Image Randomization: Shuffle the order of displayed images with a single click.
   Image Upload: Users can upload their own images to the gallery.
   Interactive Lightbox: Click on any image to open a full-screen view with navigation, download, and share options.
   Download Images: Easily download the currently viewed image from the lightbox.
   Share Images: Utilize the native Web Share API (if supported) or copy the image URL to share.
   Modern UI: Clean and intuitive interface thanks to Bootstrap and Font Awesome icons.
   Loading Spinner: Provides visual feedback during image uploads or when loading more images.

## Technologies Used

   HTML5: Structure of the web page.
   CSS3: Styling and layout, including responsive design.
   JavaScript (ES6+): Core logic for interactivity and dynamic content.
   Bootstrap 4: Responsive framework for UI components and grid system.
   Font Awesome 5: Icons for an enhanced user experience.
   Picsum Photos: Used as a source for placeholder images.

## Setup Instructions

To get this project running locally, follow these simple steps:

1.Save the files:
       Save the HTML code as `index.html`.
       Save the JavaScript code as `script.js`.
       Save the CSS code as `styles.css`.
       Ensure all three files are in the same directory.

2.Open in Browser:
       Navigate to the directory where you saved the files.
       Double-click on `index.html` to open it in your preferred web browser.

No special server setup or dependencies are required; it runs directly from your file system.

## Usage

   Browse: Scroll through the gallery to see available images.
   Search: Type keywords into the "Search images..." bar to filter the display.
   Randomize: Click the "Randomize" button to reshuffle the images.
   Upload: Click "Upload Image" to select a picture from your device and add it to the gallery.
   Load More: Click "Load More" to fetch additional placeholder images (simulated).
   View Image: Click on any thumbnail to open it in the full-screen lightbox.
   Lightbox Navigation: Use "Prev" and "Next" buttons, or the left/right arrow keys, to navigate through images in the lightbox.
   Close Lightbox: Click "Close" or press the `Esc` key to exit the lightbox.
   Download/Share: Use the respective buttons in the lightbox to save or share the image.
