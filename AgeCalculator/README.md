# ❤️ Dating App by KarthikCCodingSolutions ❤️

This is a multi-level form-based dating application concept, designed to showcase progressive user input and profile display in a clean, responsive web interface. Users can create a profile with their basic information, preferences, and photos, then view and interact with other profiles (simulated).

 ✨ Features

   Multi-Step Profile Creation: A guided form to easily create a user profile, collecting information progressively.
       Step 1: Basic Information: Name, Birthdate (with age calculation), Gender, Bio, and Interests.
       Step 2: Preferences: Likes, Dislikes, and what the user is "Looking For" in a match.
       Step 3: Photo Upload: Allows users to upload multiple pictures for their profile.
   Progress Bar: Visual feedback on the user's progress through the profile creation steps.
   Dynamic Profile Card: Once a profile is created, it's displayed as an interactive card, showcasing all the entered details and photos.
   Interactive Swiping (Simulated): Buttons to "Like" or "Dislike" profiles, simulating a dating app experience.
   Favorite Feature: Mark profiles as favorites.
   Match Management: View a list of "matched" profiles.
   Instant Chat (Simulated): A basic chat interface to interact with other profiles or matches.
   Responsive Design: Adapts to different screen sizes for a seamless experience on various devices.
   Modern UI/UX: Utilizes Font Awesome icons and a clean, appealing design for a friendly user experience.

 🚀 How to Use

1.Open `index.html`: Simply open the `index.html` file in your web browser.
2.Create Your Profile:
       Follow the on-screen steps to enter your details (Name, Birthdate, Gender, Bio, Interests, Likes, Dislikes, Looking For).
       Upload your profile pictures.
       Click "Create Profile" to complete the process.
3.Explore Profiles: Once your profile is created, you will see a simulated profile card.
       Use the "Like" (Swipe Right) or "Nope" (Swipe Left) buttons to interact with profiles.
       Click "Favorite" to mark a profile.
       Use the "Message" button to start a simulated chat with the displayed profile.
       "My Matches" will show profiles you've "matched" with.
4.  Chat: In the chat interface, type your message and click "Send" or press Enter. You'll receive a simulated response.

 🛠️ Technologies Used

   HTML5: For the structure and content of the web pages.
   CSS3: For styling and layout, including responsive design and visual enhancements.
   JavaScript (ES6+): For interactive elements, form handling, dynamic content rendering, and simulated app logic.
   Font Awesome: For scalable vector icons to enhance the user interface.
